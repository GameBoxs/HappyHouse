# Happy House
> 해당 프로젝트는 **삼성 청년 SW아카데미(Samsung SW Academy For Youth)** 1학기 최종 관통 프로젝트 결과물 입니다.

<br/>

## About
✅ 공공데이터를 활용한 부동산 매매 서비스  

✅ 매물 실 거래가, 최신 매물 정보, 지도 서비스 제공  

✅ 카카오 Map API를 활용한 위치 정보 서비스 제공  

<br/>
